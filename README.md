# Pharmacy Hub: Multi-Pharmacy Management & Medicine Tracking System## Project OverviewPharmacy Hub is a comprehensive web platform built with Flask that bridges the gap between local pharmacies and patients. It serves as a centralized marketplace where multiple pharmacies can showcase their inventory, and users can easily locate, reserve, or purchase medications online.## Core Functionalities### 1. Pharmacy Management (Vendor Side)* Inventory Control: Add, update, and manage medication lists (stock, price, descriptions).* Order Fulfillment: Track incoming orders (Pending, Processed, Completed).* Sourcing Requests: Respond to user requests for unavailable medicines.### 2. User Experience (Customer Side)* Smart Search: Real-time availability across multiple pharmacies.* Reservation & Purchase: "Book" or buy medicines directly.* Request a Medicine: Submit requests for out-of-stock items.## Technical Architecture* Backend: Python (Flask Framework).* Frontend: HTML5, CSS3, and JavaScript.* Database: Integrated tracking system for transactions and users.
