# Pharmacy Hub: Multi-Pharmacy Management & Medicine Tracking System
## Project Overview
**Pharmacy Hub** is a comprehensive web platform built with **Flask** that bridges the gap between local pharmacies and patients. It serves as a centralized marketplace where multiple pharmacies can showcase their inventory, allowing users to search, reserve, or purchase medications online.
## Core Functionalities
### 1. Pharmacy Management (Vendor Side)
 * **Inventory Control:** Add, update, and manage medication lists (stock levels, prices, and descriptions).
 * **Order Fulfillment:** Track and manage incoming orders (Pending, Processed, Completed).
 * **Sourcing Requests:** Respond to user inquiries regarding unavaiable medicines.
### 2. User Experience (Customer Side)
 * **Smart Search:** Real-time search for medicine availability across multiple registered pharmacies.
 * **Reservation & Purchase:** Securely book or buy medicines directly through the platform.
 * **Medicine Requests:** Submit special requests for out-of-stock or rare items.
## Technical Architecture
 * **Backend:** Python (Flask Framework).
 * **Frontend:** HTML5, CSS3, and JavaScript.
 * **Database:** Integrated relational database for managing transactions, inventory, and user data.
